# 🧩 ArgusOS Patches

![GitHub Workflow Status (with event)](https://img.shields.io/github/actions/workflow/status/ArgusOS/argusos-patches/release.yml)
![GPLv3 License](https://img.shields.io/badge/License-GPL%20v3-yellow.svg)

This repository contains a collection of ArgusOS Patches.

## ❓ About

Patches are small modifications to Android apps that allow you to change the behavior of or add new features,
block ads, customize the appearance, and much more.

## 💪 Features

Some of the features the patches provide are:

* 🚫 **Block ads**: Say goodbye to ads
* ⭐ **Customize your app**: Personalize the appearance of apps with various layouts and themes
* 🪄 **Add new features**: Extend the functionality of apps with lots of new features
* ⚙️ **Miscellaneous and general purpose**: Rename packages, enable debugging, disable screen capture restrictions,
  export activities, etc.
* ✨ **And much more!**

For a complete list of all available patches, visit [argusos.app/patches](https://argusos.app/patches).

## 🚀 How to get started

You can use [ArgusOS CLI](https://github.com/ArgusOS/argusos-cli) or [ArgusOS Manager](https://github.com/ArgusOS/argusos-manager) to use ArgusOS Patches.
